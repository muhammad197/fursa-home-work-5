# fursa-home-work-5
## working with terraform
- 2 Load Balancers - Internet Facing and Internal
- 4 Web Servers
- 4 Application Servers
- RDS with two nodes (MySQL)
- S3 Bucket

## how to run: 
### step 1:(init)
```sh
terraform init
```
### step 2:(plan)
```sh
terraform plan
```
### step 3:(apply)
```sh
terraform apply
```
